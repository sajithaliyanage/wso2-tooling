Modal Header

Content

  • When you create data services and resources using WSO2 Data Services Server, you have the flexibility of service enabling data stored in various types of data storages. This is done by creating datasources for each of the data storage types used by your data service.
  • Data service queries are created after creating a data source at the time a data service is created. Through data service queries, you can define different ways of extracting data in the data sources and presenting the results.
  • Data service operations are written to invoke queries. Therefore, defining operations is the 4th step in the process of creating a data service.
  • The WSO2 Data Services feature supports exposing data as a set of REST style resources in addition to the SOAP services.
